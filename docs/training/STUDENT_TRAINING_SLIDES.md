## SLIDE 4: Logging In (10 minutes)

**Title**: "How to Login to SMS"

**Content**:
- Going to SMS website
- Entering username and password
- First time login process
- Session timeout and logging out
- Forgot password procedures

**Talking Points**:
- Open your browser and go to the SMS website
- Enter your username and password
- First time you'll be asked to set a new password
- Always logout when done - it's more secure
- If you forget your password, just click "Forgot Password"

**Slide Notes**:
- Display login page URL
- Show login form fields
- Display error messages (what they mean)

**Hands-On Practice**:
1. Students open browser
2. Go to SMS website
3. Enter username and password
4. Successfully login
5. Take screenshot of dashboard

**Demo**:
- Walk through login page
- Explain each field
- Show what happens on submit
- Demonstrate successful login
- Show password reset process (don't execute)
- Show logout button and process
- Emphasize security (always logout)

**Live Interaction**:
- Walk through with each student
- Help those with difficulties
- Troubleshoot login issues

**Troubleshooting**:
- "Wrong password" - Check caps lock, try again
- "Account not found" - Check student ID spelling
- "Too many attempts" - Wait 15 minutes and retry
- "Forgot password" - Use password reset link
