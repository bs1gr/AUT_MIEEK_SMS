## ðŸš¨ Contingency Plans

### If System Goes Down During Training

1. **Immediate**: Switch to offline slides (theory only)
2. **15 min later**: Attempt restart via monitoring dashboard
3. **30 min later**: Contact support team
4. **45 min later**: Reschedule hands-on portion for next day

### If Low Attendance

- **Admins**: Mandatory, reschedule if needed
- **Teachers**: Record session, send video to no-shows
- **Students**: Offer makeup sessions during first week

### If Technical Issues

- **Wi-Fi Down**: Use mobile hotspot
- **Projector Fails**: Use individual screens
- **Demo Crashes**: Use backup test environment
