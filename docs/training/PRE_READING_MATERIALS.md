## Role-Specific Preparation

### For Administrators

**What You'll Learn**:
- User account management (create, edit, delete users)
- Role assignment and permissions
- System configuration and settings
- Report generation and analytics
- Backup and data management

**Before Training, Review**:
- Current user list (who needs accounts?)
- Role definitions (admin, teacher, student)
- Reporting requirements for your department

**Hands-On Activities**:
- Create test user accounts
- Assign roles and permissions
- Generate sample reports
- Configure system settings

---

### For Teachers

**What You'll Learn**:
- Navigating the teacher dashboard
- Entering and editing grades
- Recording student attendance
- Viewing class rosters
- Generating progress reports

**Before Training, Think About**:
- Your current grade entry process
- How you track attendance now
- Common grading scenarios (late work, makeup exams)

**Hands-On Activities**:
- Enter grades for sample students
- Record attendance for a class
- Generate progress reports
- Export data to Excel

## Common Questions (FAQ)

### General

**Q: What if I can't attend my scheduled training session?**
A: Contact your department head immediately. We'll arrange an alternative session or one-on-one training.

**Q: Will training materials be available after the session?**
A: Yes! All slides, guides, and reference materials will be emailed to you within 24 hours.

**Q: What if I forget what I learned during training?**
A: Quick reference guides will be available, plus ongoing support via email and help desk after go-live.

### Technical

**Q: Do I need to install any software?**
A: No! The SMS is web-based. You only need a modern web browser.

**Q: Can I use my smartphone for the SMS?**
A: Yes, but we recommend using a laptop or tablet for training. The mobile version works well for viewing information but data entry is easier on larger screens.

**Q: What if I have trouble with my device during training?**
A: Trainers will have backup devices available. We'll help you troubleshoot.

### Access & Security

**Q: When will I receive my real login credentials?**
A: You'll receive them via email before the system goes live. You'll use test accounts during training.

**Q: What if I forget my password after training?**
A: Use the "Forgot Password" link on the login page. You'll receive a reset link via email.

**Q: Can I access the SMS from home?**
A: Yes! The system is accessible from anywhere with internet (after go-live).

## Training Session Tips

### During Your Session

‚úì **Ask Questions** - There are no silly questions. If you're confused, others probably are too.
‚úì **Take Notes** - Write down tips specific to your workflow.
‚úì **Practice** - The more you click, the more comfortable you'll become.
‚úì **Be Patient** - Learning new software takes time. You'll get faster with practice.

### Learning Styles

- **Visual Learners**: Watch the trainer demonstrations carefully
- **Hands-On Learners**: Jump into practice exercises right away
- **Note-Takers**: Use your notepad for step-by-step procedures

### If You Get Stuck

1. **Try again** - Most issues resolve with a second attempt
2. **Ask the trainer** - They're there to help
3. **Help each other** - Collaborate with other participants
4. **Don't panic** - Everything can be undone in the test environment
