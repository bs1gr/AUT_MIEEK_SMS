## ğŸ¯ Content Development Guidelines

### Writing Style
- Clear, concise language
- Active voice preferred
- Short sentences and paragraphs
- Bullet points for lists
- Real-world examples

### Format
- Markdown (.md) for all documents
- Bilingual (EN/EL) for all content
- Include screenshots/illustrations where helpful
- Use consistent terminology across all docs
- Link between related documents

### Accessibility
- Plain language (avoid jargon)
- Provide both visual and text descriptions
- Use color + symbols (not color alone)
- Keyboard navigation instructions
- Screen reader friendly

### Translation Guidelines
- Use Greek educational system terminology
- Maintain Greek character encoding (UTF-8)
- Greek instructors review for accuracy
- Cultural context adjustments
