# Release Notes - 1.9.7

**Release Date**: 2025-11-26

## Overview

Version 1.9.3 is a housekeeping release focused on codebase modernization and legacy cleanup. This release consolidates the project's documentation and archives all pre-1.9.7 artifacts, establishing a clean foundation for future development.

## What's Changed

### Legacy Cleanup & Archival

- **Complete Legacy Archival**: All pre-1.9.7 artifacts have been moved to `archive/pre-1.9.7/`
  - 7 legacy release notes (v1.6.x, v1.8.x series)
  - Deprecated scripts (SMART_SETUP.ps1, obsolete .bat wrappers)
  - One-time migration scripts (reorganize_scripts.py, CONSOLIDATE_BAT_WRAPPERS.ps1)
  - Session documentation from development history
  - Duplicate script directories (scripts/docker/, scripts/internal/)
  - V2_QUICK_REFERENCE.md (old v2.0 roadmap document)
  - Created comprehensive README documenting all archived content

- **CHANGELOG Modernization**: Refactored to focus on v1.9.x series
  - Pre-1.9.0 history summarized and archived to `CHANGELOG_ARCHIVE.md`
  - Clean, forward-looking documentation structure
  - Proper semantic versioning links

- **Documentation Alignment**: All references updated to current infrastructure
  - Docker port standardized to 8080 across all scripts and documentation
  - Legacy script references (RUN.ps1, INSTALL.ps1, SMS.ps1) replaced with DOCKER.ps1/NATIVE.ps1
  - Installer documentation aligned with v2.0 consolidated scripts
  - Updated internal dev tools (DEVTOOLS.ps1, DIAGNOSE_STATE.ps1)
  - Updated backend monitoring router API responses

- **TODO Modernization**: Backlog refreshed for current priorities
  - Removed completed legacy phase tracking
  - Updated roadmap to focus on v1.9.x series
  - Removed outdated 1.9.7 roadmap references
