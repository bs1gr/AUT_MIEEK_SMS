# Σύστημα Διαχείρισης Φοιτητών - Γρήγορη Εκκίνηση (Ελληνικά)

**Έκδοση**: 1.17.6  
**Ημερομηνία Ενημέρωσης**: 30 Ιανουαρίου 2026  
**Γλώσσα**: Ελληνικά  
**Κοινό**: Όλοι οι χρήστες (φοιτητές, εκπαιδευτικοί, διαχειριστές)  
**Χρόνος Ανάγνωσης**: 5-10 λεπτά  

---

## 🚀 Ξεκινήστε σε 5 Λεπτά

### Βήμα 1: Πρόσβαση στο Σύστημα

**Υπολογιστής/Laptop:**
1. Ανοίξτε τον φυλλομετρητή σας (Chrome, Firefox, Safari ή Edge)
2. Πηγαίνετε στο: **https://your-institution-domain.edu/sms**
3. Θα δείτε τη σελίδα σύνδεσης του Συστήματος Διαχείρισης Φοιτητών

**Κινητή Συσκευή:**
1. Ανοίξτε τον φυλλομετρητή της συσκευής σας (ίδιοι φυλλομετρητές)
2. Πηγαίνετε στο: **https://your-institution-domain.edu/sms**
3. Το σύστημα είναι βελτιστοποιημένο για κινητά και θα προσαρμοστεί

### Βήμα 2: Σύνδεση

1. Εισάγετε το **όνομα χρήστη** ή **email** σας στο πρώτο πεδίο
2. Εισάγετε το **κωδικό πρόσβασης** στο δεύτερο πεδίο
3. Κάντε κλικ στο κουμπί **"Σύνδεση"**
4. Θα ανακατευθυνθείτε στο προσωπικό σας ταμπλό

**Πρώτη Φορά Σύνδεσης;**
- Επικοινωνήστε με το γραφείο IT του ιδρύματός σας για το αρχικό όνομα χρήστη και προσωρινό κωδικό
- Στην πρώτη σύνδεση, θα σας ζητηθεί να ορίσετε νέο κωδικό
- **Αποθηκεύστε το κωδικό σας με ασφάλεια** - θα τον χρειαστείτε

### Βήμα 3: Εξερευνήστε το Ταμπλό σας

Το ταμπλό σας εμφανίζει διαφορετικές πληροφορίες ανάλογα με το ρόλο σας:

**Για Φοιτητές:**
- Τρέχοντες βαθμοί και μέσος όρος
- Εγγεγραμμένα μαθήματα
- Αρχείο παρουσίας
- Προσχεδιασμένες εργασίες
- Μηνύματα από εκπαιδευτικούς

**Για Εκπαιδευτικούς:**
- Κατάλογος φοιτητών
- Διεπαφή εισαγωγής βαθμών
- Παρακολούθηση παρουσίας
- Υλικό μαθήματος
- Εργαλεία αναφοράς

**Για Διαχειριστές:**
- Κατάσταση συστήματος
- Επιλογές διαχείρισης χρηστών
- Δημιουργία αναφορών
- Διαμόρφωση συστήματος

- Χρησιμοποιείτε απλούς κωδικούς (123456, password κτλ.)
- Κάνετε κλικ σε ύποπτους συνδέσμους από emails

## 📱 Απαιτήσεις Φυλλομετρητή

**Ελάχιστες Απαιτήσεις:**

- JavaScript ενεργοποιημένο
- Cookies ενεργοποιημένα

**Συνιστώμενη Ρύθμιση:**
- Τελευταία έκδοση του φυλλομετρητή
- Καθαρισμός κρυφής μνήμης κάθε μήνα
- Απενεργοποίηση επεκτάσεων που ενδέχεται να παρεμβαίνουν

## 🆘 Ξέχασα τον Κωδικό μου

1. Κάντε κλικ στο **"Ξέχασα τον Κωδικό;"** στη σελίδα σύνδεσης
2. Εισάγετε το email ή όνομα χρήστη σας
3. Ελέγξτε το email σας για σύνδεσμο επαναφοράς (εντός 10 λεπτών)
4. Κάντε κλικ στο σύνδεσμο και δημιουργήστε νέο κωδικό
5. Συνδεθείτε με το νέο κωδικό

**Αν δεν λάβατε email επαναφοράς:**
- Ελέγξτε το φάκελο spam/ανεπιθύμητα
- Περιμένετε 10 λεπτά και δοκιμάστε ξανά
- Επικοινωνήστε με το γραφείο IT

---

## 🌍 Ρυθμίσεις Γλώσσας

Το σύστημα υποστηρίζει **Ελληνικά** και **Αγγλικά**:

1. Μετά τη σύνδεση, αναζητήστε τον επιλογέα γλώσσας (συνήθως πάνω δεξιά)
2. Κάντε κλικ στο αναπτυσσόμενο μενού γλώσσας
3. Επιλέξτε **English** (EN) ή **Ελληνικά** (EL)
4. Η διεπαφή θα αλλάξει αμέσως

Η προτίμησή σας για γλώσσα αποθηκεύεται αυτόματα.

## 💾 Πώς να Χρησιμοποιείτε την Αναζήτηση

**Εύρεση Φοιτητών/Μαθημάτων/Βαθμών:**

1. Αναζητήστε το πλαίσιο αναζήτησης (συνήθως πάνω)
2. Πληκτρολογήστε το όνομα, ID ή κωδικό
3. Τα αποτελέσματα εμφανίζονται αυτόματα
4. Κάντε κλικ στο αποτέλεσμα για λεπτομέρειες

**Συμβουλές Αναζήτησης:**
- Η αναζήτηση δεν είναι ευαίσθητη σε μεγάλα/μικρά (JOHN = john)
- Μερικά ονόματα λειτουργούν (πληκτρολογήστε "Γιαν" για "Γιάννης")
- Χρησιμοποιήστε εισαγωγικά για ακριβή ταίριασμα: "Γιάννης Παπαδόπουλος"
- Χρησιμοποιήστε φίλτρα παράλληλα με αναζήτηση

## 📊 Προβολή Δεδομένων

### Βαθμοί
- **Φοιτητές**: Δείτε τους τελικούς βαθμούς και GPA
- **Εκπαιδευτικοί**: Δείτε υποβολές και εισάγετε βαθμούς
- **Διαχ.**: Πρόσβαση σε όλα τα αρχεία βαθμών

### Παρουσία
- **Φοιτητές**: Ελέγξτε το ποσοστό παρουσίας
- **Εκπαιδευτικοί**: Σημειώστε παρουσία
- **Διαχ.**: Δημιουργήστε αναφορές παρουσίας

### Μαθήματα
- **Φοιτητές**: Περιηγηθείτε, εγγραφείτε
- **Εκπαιδευτικοί**: Δείτε κατάλογο φοιτητών
- **Διαχ.**: Δημιουργήστε/διαχειριστείτε μαθήματα

## 🖨️ Εξαγωγή και Εκτύπωση

**Πώς να Εξάγετε Δεδομένα:**

1. Βρείτε τα δεδομένα που θέλετε να εξάγετε
2. Αναζητήστε κουμπί **"Εξαγωγή"** (συνήθως πάνω δεξιά)
3. Επιλέξτε μορφή:
   - **PDF** - για προβολή και εκτύπωση
   - **Excel** - για ανάλυση
   - **CSV** - για άλλα συστήματα
4. Κάντε κλικ **"Λήψη"** ή **"Εξαγωγή"**
5. Το αρχείο θα κατέβει στον υπολογιστή σας

**Πώς να Εκτυπώσετε:**
1. Δείτε τη σελίδα ή εξάγετε σε PDF
2. Πατήστε **Ctrl+P** (Windows) ή **Cmd+P** (Mac)
3. Επιλέξτε εκτυπωτή και ρυθμίσεις
4. Κάντε κλικ **"Εκτύπωση"**
