# ⚡ Γρήγορη Εκκίνηση - Student Management System

> Ενοποιημένα Σκριπτ (v2.0)
>
> - Χρησιμοποιήστε: **DOCKER.ps1** (Docker fullstack) ή **NATIVE.ps1** (Native dev mode).
> - Τα παλιά RUN.ps1 / SMS.ps1 / INSTALL.ps1 / SMART_SETUP.ps1 έχουν αποσυρθεί.

## 🚀 Σε 3 Βήματα

### Βήμα 1: Προαπαιτούμενα (Μόνο πρώτη φορά)

```powershell
# Εγκατάσταση Python 3.13+ και Node.js 18+

# Κατεβάστε από:
# - Python: https://www.python.org/downloads/

# - Node.js: https://nodejs.org/

```text
### Βήμα 2: Αρχική Εγκατάσταση (Μόνο πρώτη φορά)

```powershell
# Clone το repository

git clone https://github.com/bs1gr/AUT_MIEEK_SMS.git
cd AUT_MIEEK_SMS

# Επιλογή 1 (Docker fullstack – προτεινόμενο):

.\DOCKER.ps1 -Install

# Επιλογή 2 (Native ανάπτυξη):

.\NATIVE.ps1 -Setup

```text
### Βήμα 3: Καθημερινή Χρήση

```powershell
# Docker (fullstack container)

.\DOCKER.ps1 -Start     # Εκκίνηση / επανεκκίνηση
.\DOCKER.ps1 -Update    # Γρήγορο rebuild
.\DOCKER.ps1 -Stop      # Διακοπή

# Native (ξεχωριστό backend + frontend)

.\NATIVE.ps1 -Start
.\NATIVE.ps1 -Stop

```text
**Αυτό είναι όλο!** 🎉
