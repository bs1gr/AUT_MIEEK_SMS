INSTALLATION OPTIONS:
=====================

Option 1: Single PC (SQLite)
  - Fastest setup (< 1 minute)
  - No QNAP needed
  - Data stored locally
  - Good for: Individual users, teachers, labs

Option 2: Team Deployment (QNAP)
  - Shared database
  - Real-time data sync
  - Same users as main SMS
  - Good for: Districts, schools, offices
  - Setup time: 5-10 minutes per PC

Option 3: Automated Deployment (Group Policy)
  - Deploy to many PCs automatically
  - Pre-configured credentials (optional)
  - Centralized management
  - Good for: Large organizations
