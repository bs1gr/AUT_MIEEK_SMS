# ⚡ Γρήγορη Εκκίνηση - Student Management System

> Παρωχημένα βοηθητικά σκριπτ
>
> - Τα START.bat και ONE-CLICK.ps1/ONE-CLICK.bat είναι παρωχημένα και δεν υποστηρίζονται πλέον.
> - Προτιμήστε: RUN.ps1 (εκκίνηση) και SMS.ps1 (διαχείριση).
> - Για πλήρη εγκατάσταση και εκκίνηση, χρησιμοποιήστε το RUN.ps1 ή το SMART_SETUP.ps1.

## 🚀 Σε 3 Βήματα

### Βήμα 1: Προαπαιτούμενα (Μόνο πρώτη φορά)

```powershell
# Εγκατάσταση Python 3.13+ και Node.js 18+
# Κατεβάστε από:
# - Python: https://www.python.org/downloads/
# - Node.js: https://nodejs.org/
```

### Βήμα 2: Αρχική Εγκατάσταση (Μόνο πρώτη φορά)

```powershell
# Clone το repository
git clone https://github.com/bs1gr/AUT_MIEEK_SMS.git
cd AUT_MIEEK_SMS

# Αυτόματη εγκατάσταση και εκκίνηση
.\RUN.ps1
```

### Βήμα 3: Καθημερινή Χρήση

```powershell
# Κάθε φορά που θέλετε να εκκινήσετε το σύστημα
.\RUN.ps1
```

**Αυτό είναι όλο!** 🎉

## ⚠️ Βασική Αντιμετώπιση Προβλημάτων

### Port 8000 κατειλημμένο

```powershell
# Βρείτε ποια διεργασία χρησιμοποιεί τη θύρα
netstat -ano | findstr :8000

# Προτιμήστε την ασφαλή ροή χειρισμού:
#  .\scripts\maintenance\stop_frontend_safe.ps1 -ControlUrl 'http://127.0.0.1:8000'
# Αν είναι απολύτως απαραίτητο και είστε χειριστής, τρέξτε την έκτακτη ρουτίνα με επιβεβαίωση:
#  .\scripts\internal\KILL_FRONTEND_NOW.ps1 -Confirm
```

### Λείπουν modules

```powershell
cd backend
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### Frontend δεν φορτώνει

```powershell
cd frontend
npm install
npm run dev
```
